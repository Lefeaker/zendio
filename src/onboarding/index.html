欢迎使用 Zendio

让我们快速配置您的剪藏插件

感谢您安装 Zendio！这个引导将帮助您快速配置插件，让您能够轻松地将网页内容和 AI 对话保存到 Obsidian 中。

1

配置 Obsidian Local REST API

首先需要在 Obsidian 中安装并配置 Local REST API 插件，这是连接插件与 Obsidian 的桥梁。

  • 在 Obsidian 中安装并启用 "Local REST API" 插件
  • 打开插件中的 Enable Non-encrypted (HTTP) Server
  • 记录 HTTPS URL（通常是 https://127.0.0.1:27124）
  • 记录 HTTP URL（通常是 http://127.0.0.1:27123）
  • 记录您的 Obsidian 仓库名称，copy Local REST API 的 API 密钥
  • 将以上信息填入插件里，并进行连接测试
2

配置额外仓库（可选）

如果您有多个 Obsidian 仓库，可以配置额外仓库并设置路由规则，让不同类型的内容自动保存到对应的仓库中。

  • 仓库配置方法和上面一致
  • 可根据域名、关键词或 URL 模式设置路由规则：将特定域名或含有特定关键词的内容保存到额外仓库中
  • 不符合规则的内容会保存到默认仓库
3

主要功能

让我们快速了解一下插件的主要功能，帮助您更好地使用插件。

网页剪藏
  • 点击网页空白处，即可剪藏整个网页（请等待网页加载完毕，并滚动网页使得图片资源完全加载）
  • 自动识别主流 AI 聊天对话，格式化保存 AI 对话记录
剪藏/阅读模式
  • 选中文本后右键，或打开辅助键后，按下辅助键选择内容，即可对内容进行评论，并将选中内容与批注一起保存到obsidian里
  • 亦可进入阅读模式，在同一页面中选取多个片段进行评论，一同保存到obsidian里
  • 阅读模式中根据页面排布，自动排列文字片段顺序，点击选中文字的序号，一键定位
  • 阅读模式可保存全文到 obsidian 里，选中内容以高亮形式显示
  • 无论片段还是阅读模式选中内容，保存页面精准链接，随时一键返回网页位置
  • 在 obsidian 里安装 sidebar highlights 插件，查看批注内容更方便
视频模式
  • YouTube 或 bilibili 适配播放页面，打开视频模式，随时记录视频时间点并添加笔记
  • 点击时间点旁边的序号，一键定位反复观看
  • 勾选页面文字，精彩评论，一键捕捉
  • 保存到obsidian 后，随时一键回溯视频精准时间点
4

辅助功能

插件还提供了多种辅助功能，让您的使用体验更加便捷。

  • 同一设备多个浏览器，一键复制当前配置，粘贴同步，无需重复设置
  • 域名映射：将常用网站映射为友好的文件夹名称
  • 自定义路径配置，根据需要调整路径
  • 智能诊断：配置有问题？智能诊断，快速排查
