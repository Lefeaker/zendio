## 通用验证步骤

在测试每个站点前，请在 DevTools Console 执行:
```js
globalThis.__aiobReaderDialogVersion = 'daisy';
```

### 验证清单

- [ ] **Panel 出现**: 选中文本后 Panel 正确显示
- [ ] **样式隔离**: Daisy 样式未被站点 CSS 覆盖
- [ ] **焦点管理**: Tab 在 Dialog 内循环，Esc 关闭正常
- [ ] **高亮列表**: 展开/折叠、编辑功能正常
- [ ] **导出流程**: Finish 按钮触发导出，成功/失败提示正常
