# Production Code Hotspots

Last updated: 2026-05-22

`npm run audit:production-shape:report` now enforces the final hotspot thresholds below. The audit fails if a listed facade exceeds its LOC budget or if the YAML widget / Stitch renderer adapter reintroduces direct visible text assignments instead of schema or i18n-backed content. Large post-split owners remain tracked here even when the safe decision is to retain and budget them instead of deleting or splitting during release closure.
