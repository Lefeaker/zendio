# Retired Code Inventory

Last updated: 2026-05-10

## Decision Rules

- `delete-now`: path is not in the production build graph, has no runtime import, and tests are either removed or migrated.
- `migrate-then-delete`: path currently has test, script, or internal callers that must move to Stitch/domain code first.
- `retain-production`: path participates in production build or is the current domain owner.
- `retain-test-fixture`: path is not production but is an explicit fixture outside `src`.
