## 🚨 关键问题分析

### 1. **Assets 目录位置问题** 🔴 高优先级

**原计划问题**：
```
❌ 原计划：assets/ → src/assets/
❌ 原计划：_locales/ → src/assets/locales/
```

**问题分析**：
- **构建复杂化**：`scripts/build.mjs` 需要大幅修改路径
- **引用路径变深**：HTML 文件中的相对路径从 `../assets/` 变为 `../../assets/`
- **Manifest 路径混乱**：扩展清单中的资源引用需要大量调整
- **开发体验下降**：静态资源路径变得不直观

**修正方案**：
```
✅ 修正后：assets/ → assets/（保持不变）
✅ 修正后：_locales/ → _locales/（保持不变）
```

### 2. **遗漏的路径更新项目** 🟡 中优先级

**新发现的 15 个遗漏项目**：

#### HTML 文件资源引用（5 项）
- `src/options/index.html` 中的 CSS 样式引用
- `src/options/index.html` 中的图标资源引用

#### Manifest 配置路径（2 项）
- 背景脚本路径：`background/index.js` → `core/background/index.js`
- 选项页面路径：`options/index.html` → `core/options/index.html`

#### 构建脚本路径（6 项）
- 入口点路径更新（3 个）
- 样式文件路径更新（2 个）
- HTML 文件路径更新（1 个）

#### 配置文件（2 项）
- `.gitignore` 需要添加新目录
- 国际化文件路径确认
