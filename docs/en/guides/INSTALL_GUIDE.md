# All in Ob - 安装指南

## 📦 获取扩展文件

你应该已经收到了一个 `.zip` 文件，例如 `all-in-ob-v0.2.0.zip`

## 🔧 安装步骤

### 方法一：直接加载解压后的文件夹（推荐）

1. **解压文件**
   - 将收到的 `.zip` 文件解压到一个固定的位置
   - ⚠️ 注意：不要删除解压后的文件夹，扩展需要一直从这个位置运行

2. **打开 Chrome 扩展管理页面**
   - 打开 Chrome 浏览器
   - 在地址栏输入：`chrome://extensions/`
   - 或者：点击右上角三个点 → 更多工具 → 扩展程序

3. **启用开发者模式**
   - 在扩展管理页面右上角，找到"开发者模式"开关
   - 将其打开（变成蓝色）

4. **加载扩展**
   - 点击左上角的"加载已解压的扩展程序"按钮
   - 选择刚才解压的文件夹
   - 点击"选择"

5. **完成！**
   - 扩展应该会出现在扩展列表中
   - 你会在浏览器工具栏看到 All in Ob 的图标

### 方法二：拖拽安装（某些 Chrome 版本支持）

1. **打开扩展管理页面**
   - 在地址栏输入：`chrome://extensions/`
   - 启用"开发者模式"

2. **拖拽安装**
   - 将 `.zip` 文件直接拖拽到扩展管理页面
   - Chrome 会自动解压并安装

## ⚙️ 配置扩展

安装完成后，需要配置 Obsidian 连接：

1. **点击扩展图标**
   - 在浏览器工具栏找到 All in Ob 图标
   - 右键点击图标，选择"选项"

2. **配置 Obsidian 连接**
   - 在设置页面中配置你的 Obsidian 仓库信息
   - 确保 Obsidian 的 Local REST API 插件已启动
   - 测试连接是否成功

## 🔄 更新扩展

当收到新版本的 `.zip` 文件时：

1. 解压新版本文件到新的文件夹
2. 在 `chrome://extensions/` 页面
3. 点击旧版本扩展的"移除"按钮
4. 按照上述安装步骤加载新版本

或者：

1. 直接用新版本文件覆盖旧文件夹中的内容
2. 在扩展管理页面点击"重新加载"按钮（圆形箭头图标）

## ❓ 常见问题

### 扩展显示为灰色/无法使用

- 确保 Obsidian 的 Local REST API 插件已安装并启动
- 检查扩展设置中的连接配置是否正确
- 尝试在设置页面点击"测试连接"

### 扩展突然不工作了

- 检查解压的文件夹是否被移动或删除
- 在扩展管理页面点击"重新加载"按钮
- 如果问题持续，尝试移除并重新安装

### Chrome 提示"此扩展可能已损坏"

- 这是因为扩展不是从 Chrome 应用商店安装的
- 这是正常的，点击"保留"即可
- 每次 Chrome 启动时可能会提示，这是 Chrome 的安全机制

### 如何固定扩展图标

- 在浏览器工具栏右侧，点击拼图图标
- 找到 All in Ob
- 点击图钉图标将其固定到工具栏

## 📝 使用说明

安装配置完成后，你可以：

- 在任何网页上点击扩展图标，将内容剪藏到 Obsidian
- 右键选中的文本，通过右键菜单快速剪藏
- 在支持的 AI 对话平台（ChatGPT、Claude、Gemini 等）上剪藏对话内容

详细使用说明请参考 [README.md](../../../README.md)

## 🆘 需要帮助？

如果遇到问题，请：

1. 查看浏览器控制台的错误信息（F12 → Console）
2. 检查扩展的后台页面错误（在扩展管理页面点击"错误"）
3. 联系扩展开发者获取支持

---

**注意事项：**

- ⚠️ 不要删除解压后的扩展文件夹
- ⚠️ 每次 Chrome 更新后，可能需要重新启用开发者模式
- ⚠️ 某些企业或学校的 Chrome 可能禁用了开发者模式，无法安装此类扩展
