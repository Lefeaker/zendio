# 快速开始指南

## 🎯 5 分钟快速上手

### 步骤 1: 加载扩展 (1 分钟)

1. 打开 Chrome 浏览器
2. 地址栏输入：`chrome://extensions/`
3. 打开右上角的 **"开发者模式"**
4. 点击 **"加载已解压的扩展程序"**
5. 选择文件夹：`AiiinOB/your-extension/dist`
6. 看到扩展图标出现 ✅

### 步骤 2: 配置 Obsidian (2 分钟)

1. 打开 Obsidian
2. 设置 → 第三方插件 → 浏览
3. 搜索并安装：**"Local REST API"**
4. 启用插件
5. 在插件设置中：
   - 复制 **API 地址**（如：`https://127.0.0.1:27124`）
   - 复制 **API 密钥**
   - 确保插件状态为 **绿色**（已启动）

### 步骤 3: 配置扩展 (2 分钟)

1. 在 Chrome 中右键点击扩展图标
2. 选择 **"选项"**
3. 填写配置：
   ```
   Base URL: https://127.0.0.1:27124
   Vault Name: 你的库名（如：MyVault）
   API Key: 粘贴从 Obsidian 复制的密钥
   ```
4. 点击 **"保存配置"**
5. 看到成功提示 ✅

### 步骤 4: 测试剪藏 (30 秒)

#### 测试 AI 对话：

1. 打开 https://chat.openai.com/ 或 https://claude.ai/
2. 进行一段简单对话
3. 右键点击页面 → **"剪藏到 Obsidian（后台）"**
4. 等待桌面通知
5. 在 Obsidian 中查看 `AI/` 文件夹 ✅

#### 测试文章剪藏：

1. 打开任意技术博客或新闻文章
2. 右键点击页面 → **"剪藏到 Obsidian（后台）"**
3. 等待桌面通知
4. 在 Obsidian 中查看 `Clippings/` 文件夹 ✅

## 🔧 常见问题

### ❌ 证书错误

**问题：** 剪藏失败，提示证书错误

**解决：**

1. 在浏览器中访问：`https://127.0.0.1:27124`
2. 点击 **"高级"** → **"继续访问"**
3. 接受自签名证书
4. 重试剪藏

### ❌ 找不到库

**问题：** 提示 "Vault not found"

**解决：**

1. 检查 Vault Name 是否与 Obsidian 中的库名完全一致
2. 注意大小写和空格
3. 在 Obsidian 设置中确认库名

### ❌ API 密钥错误

**问题：** 提示 "Unauthorized"

**解决：**

1. 重新复制 Obsidian 中的 API 密钥
2. 确保没有多余的空格
3. 重新保存扩展配置
