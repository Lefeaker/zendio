# UI 优化测试指南

## 🚀 快速开始

### 1. 构建项目
```bash
cd AiiinOB/your-extension
npm run build
```

### 2. 加载扩展到浏览器

#### Chrome / Edge
1. 打开 `chrome://extensions/` 或 `edge://extensions/`
2. 启用右上角的"开发者模式"
3. 点击"加载已解压的扩展程序"
4. 选择 `AiiinOB/your-extension/dist` 目录

#### Firefox
1. 打开 `about:debugging#/runtime/this-firefox`
2. 点击"临时载入附加组件"
3. 选择 `AiiinOB/your-extension/dist/manifest.json`

## 🐛 常见问题排查

### 问题 1：CSS 文件未加载
**症状：** 页面显示为默认样式，没有渐变效果

**解决方法：**
1. 检查 `dist/styles/` 目录是否存在
2. 检查 CSS 文件是否正确复制
3. 重新运行 `npm run build`
4. 检查浏览器控制台是否有 404 错误

### 问题 2：Logo 未显示
**症状：** 设置页面顶部没有 Logo

**解决方法：**
1. 检查 `dist/assets/logo/options-logo.png` 是否存在
2. 检查文件权限
3. 重新运行 `npm run build`

### 问题 3：渐变效果不显示
**症状：** 标题和按钮没有渐变效果

**解决方法：**
1. 检查浏览器是否支持 CSS 渐变
2. 检查 CSS 变量是否正确加载
3. 打开浏览器开发者工具检查元素样式

### 问题 4：Focus 发光环不显示
**症状：** 输入框 Focus 时没有紫色发光环

**解决方法：**
1. 检查浏览器是否支持 box-shadow
2. 检查 CSS 变量 `--ring` 是否正确
3. 检查是否有其他样式覆盖

### 问题 5：动画不流畅
**症状：** 按钮悬浮或弹窗出现时动画卡顿

**解决方法：**
1. 检查浏览器性能
2. 关闭浏览器硬件加速试试
3. 检查是否有其他扩展冲突
