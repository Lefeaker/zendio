# UI 视觉效果预览

## 🎨 设计风格概览

### 核心设计理念

- **深色霓虹主题**：深邃的背景 + 紫青渐变点缀
- **精致细节**：柔和阴影、统一圆角、环形发光
- **流畅动画**：微妙的悬浮效果、平滑的过渡
- **清晰层级**：三层背景色、明确的视觉层次

4. 缓动：cubic-bezier(0.2, 0.8, 0.2, 1)

## 🌟 设计亮点

### 1. 霓虹渐变

- **应用场景**：标题、按钮、分隔线
- **渐变方向**：135°（左上到右下）
- **颜色过渡**：紫 → 靛 → 青
- **视觉效果**：科技感、未来感

### 2. 环形发光

- **应用场景**：输入框 Focus、按钮悬浮
- **发光颜色**：紫色（rgba(124, 58, 237, 0.35)）
- **发光范围**：3px
- **视觉效果**：精致、高级

### 3. 柔和阴影

- **应用场景**：卡片、按钮、弹窗
- **阴影参数**：0 4px-10px 16px-30px rgba(0,0,0,0.25-0.35)
- **视觉效果**：层次感、立体感

### 4. 微妙动画

- **悬浮效果**：上移 1px
- **饱和度提升**：110%
- **过渡时间**：120-200ms
- **视觉效果**：流畅、自然
