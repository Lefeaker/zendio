# 右键菜单问题诊断

## 问题说明

Chrome 扩展的右键菜单有两种情况：

### 1. 右键点击扩展图标（工具栏）

- ❌ **无法自定义** - 这是 Chrome 浏览器自带的菜单
- 显示的是：管理扩展、从 Chrome 中移除、固定/取消固定等选项
- 这个菜单**无法通过代码修改**，是 Chrome 的限制

### 2. 右键点击网页内容

- ✅ **可以自定义** - 通过 `chrome.contextMenus` API
- 这是你的扩展已经实现的功能
- 包括：
  - 在页面空白处右键 → "剪藏整个页面到 Obsidian"
  - 选中文字后右键 → "✂️ 剪藏选中内容到 Obsidian"

## 正确的使用方式

### 方式 1: 左键点击扩展图标

- **操作**：左键点击工具栏中的扩展图标
- **效果**：剪藏当前整个页面
- **实现**：通过 `chrome.action.onClicked` 监听器

### 方式 2: 在网页中右键

- **操作**：在网页内容区域右键点击
- **效果**：显示自定义菜单
  - 空白处右键 → "剪藏整个页面到 Obsidian"
  - 选中文字右键 → "✂️ 剪藏选中内容到 Obsidian"
- **实现**：通过 `chrome.contextMenus` API

## 诊断步骤

### 步骤 1: 检查扩展是否正确加载

1. 打开 `chrome://extensions/`
2. 找到 "Zendio"
3. 确保：
   - ✅ 扩展已启用（开关是蓝色的）
   - ✅ 没有错误提示
   - ✅ 显示版本号 0.2.0

### 步骤 2: 重新加载扩展

1. 在 `chrome://extensions/` 页面
2. 点击扩展卡片上的 **"重新加载"** 按钮（圆形箭头图标）
3. 等待几秒钟

### 步骤 3: 测试左键点击扩展图标

1. 打开任意网页（不要是 chrome:// 开头的页面）
2. **左键点击**工具栏中的扩展图标
3. 观察是否：
   - 出现通知："已保存到 Obsidian"
   - 或者出现错误通知

### 步骤 4: 测试网页右键菜单

1. 打开测试页面：`tests/manual/test-context-menu.html`
2. 在页面空白处右键
3. 查看是否有 "剪藏整个页面到 Obsidian" 菜单项
4. 选中一些文字，右键
5. 查看是否有 "✂️ 剪藏选中内容到 Obsidian" 菜单项

### 步骤 5: 检查浏览器控制台

1. 打开任意网页
2. 按 F12 打开开发者工具
3. 切换到 "Console" 标签
4. 尝试点击扩展图标或右键菜单
5. 查看是否有错误信息

### 步骤 6: 检查 Background Script 日志

1. 打开 `chrome://extensions/`
2. 找到 "Zendio"
3. 点击 "service worker" 链接（在 "检查视图" 下方）
4. 会打开一个新的开发者工具窗口
5. 查看 Console 中的日志信息
6. 尝试触发剪藏功能，观察日志输出

## 常见问题

### Q1: 为什么右键点击扩展图标没有"剪藏"选项？

**A**: 这是 Chrome 的设计限制。右键点击扩展图标显示的是浏览器自带的管理菜单，无法自定义。

**解决方案**：

- 使用**左键点击**扩展图标来剪藏整个页面
- 或者在网页内容区域右键，使用自定义菜单

### Q2: 左键点击扩展图标没有反应？

**可能原因**：

1. 扩展没有正确加载
2. 当前页面不支持扩展（如 chrome:// 页面）
3. Background script 出错

**解决方案**：

1. 重新加载扩展
2. 切换到普通网页（如 https://www.baidu.com）
3. 检查 Background Script 控制台是否有错误

### Q3: 网页右键菜单不显示？

**可能原因**：

1. 扩展没有正确加载
2. contextMenus 没有正确创建
3. 页面刷新后扩展状态丢失

**解决方案**：

1. 重新加载扩展
2. 刷新网页（F5）
3. 检查 Background Script 控制台

## 技术说明

### Chrome 扩展的两种交互方式

#### 1. Action (扩展图标)

```json
{
  "action": {
    "default_title": "Clip to Obsidian"
  }
}
```

- 没有 `default_popup` → 点击触发 `chrome.action.onClicked` 事件
- 有 `default_popup` → 点击打开弹窗页面

#### 2. Context Menus (右键菜单)

```javascript
chrome.contextMenus.create({
  id: 'clip-current',
  title: '剪藏整个页面到 Obsidian',
  contexts: ['page']
});
```

- 只能在**网页内容区域**显示
- 不能在扩展图标上显示

## 总结

✅ **正确的使用方式**：

1. **左键点击扩展图标** → 剪藏整个页面
2. **网页空白处右键** → 选择"剪藏整个页面到 Obsidian"
3. **选中文字后右键** → 选择"✂️ 剪藏选中内容到 Obsidian"

❌ **不支持的操作**：

- 右键点击扩展图标（这是 Chrome 的限制）
