# 无痕模式启用指南

## 问题现象
插件在无痕窗口中显示为灰色，无法使用。

## 解决方案

### 步骤 1：打开扩展管理页面
1. 在 Chrome 地址栏输入：`chrome://extensions/`
2. 或者点击右上角的扩展图标 → "管理扩展"

### 步骤 2：找到你的插件
找到 **"All in Obsidian"** 扩展

### 步骤 3：启用无痕模式权限
1. 点击扩展卡片上的 **"详细信息"** 按钮
2. 向下滚动找到 **"在无痕模式下启用"** 选项
3. **打开这个开关** ✅

### 步骤 4：测试
1. 打开一个无痕窗口（Ctrl/Cmd + Shift + N）
2. 确认扩展图标不再是灰色
3. 测试剪藏功能

## 重要说明

### 关于 `incognito: "spanning"` 配置
- 我们已经在 `manifest.json` 中设置了 `"incognito": "spanning"`
- 这意味着普通窗口和无痕窗口会**共享配置和存储**
- 你只需要配置一次 Obsidian 连接信息

### 为什么需要手动启用？
Chrome 出于安全考虑，默认不允许扩展在无痕模式下运行。即使扩展的 manifest 中声明了 `incognito` 配置，用户仍然需要在扩展管理页面手动授权。

## 截图参考位置
在扩展详情页面中，你会看到类似这样的选项：
```
□ 允许访问文件网址
☑ 在无痕模式下启用  ← 确保这个选项被勾选
□ 允许在所有网站上
```

## 如果还是不行
1. 确认扩展已经重新加载
2. 完全关闭所有无痕窗口后重新打开
3. 检查 Chrome 版本是否支持 Manifest V3
4. 查看扩展的错误日志（在扩展管理页面点击"错误"）
