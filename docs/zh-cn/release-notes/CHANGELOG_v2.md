# 更新日志 - v2.0

## 🎉 版本 2.0 - 2024-01-15

### ✨ 新功能

#### 1. 🌐 域名映射功能
- **核心功能**：将网站域名映射为友好的中文名称
- **预设映射**：内置常用网站映射（公众号、知乎、知识星球等）
- **自定义映射**：支持添加、编辑、删除自定义映射
- **智能应用**：自动在路径生成时应用映射

**使用示例：**
```
映射前：Clippings/mp.weixin.qq.com/2024/article.md
映射后：Clippings/公众号/2024/article.md
```

**预设映射列表：**
- `mp.weixin.qq.com` → 公众号
- `wx.zsxq.com` → 知识星球
- `www.zhihu.com` → 知乎
- `juejin.cn` → 掘金
- `www.youtube.com` → YouTube
- `twitter.com` / `x.com` → Twitter
- `medium.com` → Medium

#### 2. 🎨 全新 UI 设计
- **深色主题**：采用现代化的深色配色方案
- **视觉层次**：清晰的区块划分和视觉引导
- **图标系统**：为每个功能区添加直观的图标
- **响应式布局**：适配不同屏幕尺寸

#### 3. 📝 改进的模板配置
- **变量提示**：每个模板输入框下方显示可用变量
- **实时预览**：更清晰的模板格式说明
- **示例展示**：提供常用模板示例

#### 4. 🔍 增强的诊断功能
- **详细信息**：显示完整的配置详情
- **连接测试**：同时测试 HTTPS 和 HTTP 连接
- **问题检测**：自动检测常见配置问题
- **一键修复**：智能修复配置错误

### ⚠️ 注意事项

#### 兼容性
- ✅ 完全向后兼容 v1.x 配置
- ✅ 不使用映射时行为与旧版本一致
- ✅ 现有的剪藏内容不受影响

#### 配置迁移
- 旧配置会自动保留
- 新增的 `domainMappings` 字段为可选
- 不配置映射时使用原始域名

#### 路径变化
- 使用映射后，文件路径会改变
- 建议在正式使用前测试路径效果
- 可以随时修改或删除映射
