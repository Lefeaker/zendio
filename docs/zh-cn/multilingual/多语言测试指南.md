# 多语言功能测试指南

## 测试前准备

### 1. 重新加载扩展

1. 打开 `chrome://extensions/`
2. 找到 "All in Obsidian" 扩展
3. 点击"重新加载"按钮 🔄

### 2. 打开设置页面

- 方法1：右键点击扩展图标 → 选择"选项"
- 方法2：在 `chrome://extensions/` 页面点击"详细信息" → "扩展程序选项"

## 常见问题排查

### 问题 1: 切换语言后部分文字没有更新

**可能原因：**
- 该元素没有添加 `data-i18n` 属性
- 动态生成的元素需要手动设置文本

**解决方法：**
- 检查 HTML 元素是否有 `data-i18n` 属性
- 检查 TypeScript 代码是否使用 `getMessages()` 获取翻译

### 问题 2: 右键菜单没有更新

**可能原因：**
- 右键菜单在扩展安装时创建，需要重新加载扩展

**解决方法：**
- 打开 `chrome://extensions/`
- 点击扩展的"重新加载"按钮

### 问题 3: 通知消息显示英文

**可能原因：**
- 代码中硬编码了英文字符串
- 没有使用 `getMessages()` 获取翻译

**解决方法：**
- 检查 background script 中的通知代码
- 确保使用 `await getMessages()` 获取翻译

### 问题 4: 语言设置没有保存

**可能原因：**
- `chrome.storage.sync` 权限问题
- 存储配额已满

**解决方法：**
- 检查 manifest.json 中是否有 "storage" 权限
- 清理不必要的存储数据
