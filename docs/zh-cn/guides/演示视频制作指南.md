# 📹 选中内容剪藏功能 - 演示视频制作指南

## 目的

制作一个简短的演示视频或 GIF，展示选中内容剪藏功能的使用流程。

---

## 🎬 场景 1：基础剪藏流程（30 秒）

### 准备工作

- 打开一个技术博客页面（如 Medium 文章）
- 确保 Obsidian 和 Local REST API 已运行
- 准备好屏幕录制工具

### 录制步骤

**第 1 步：选中文本（3 秒）**

```
动作：用鼠标拖动选中一段有价值的文本（1-2 段落）
提示：选中的文本会高亮显示
```

**第 2 步：右键菜单（2 秒）**

```
动作：右键点击选中的文本
提示：菜单中会显示 "✂️ 剪藏选中内容到 Obsidian"
```

**第 3 步：对话框出现（2 秒）**

```
动作：点击菜单项
效果：优雅的对话框弹出，显示选中内容预览
```

**第 4 步：添加评论（10 秒）**

```
动作：在评论框中输入
示例文本：
  💡 这个观点很有启发性，可以应用到我的项目中
```

**第 5 步：确认剪藏（2 秒）**

```
动作：点击 "✂️ 剪藏" 按钮
效果：对话框关闭
```

**第 6 步：通知显示（3 秒）**

```
效果：浏览器右上角显示通知
内容：已保存到 Obsidian
      Clippings/medium.com/2025/2025-10-02/article-title-@142345.md
```

**第 7 步：在 Obsidian 中查看（8 秒）**

```
动作：切换到 Obsidian
效果：
  1. 文件已自动创建
  2. 显示剪藏的内容
  3. 显示添加的评论
```

## 🎬 场景 2：精确定位演示（20 秒）

### 录制步骤

**第 1 步：打开剪藏文件（3 秒）**

```
动作：在 Obsidian 中打开刚才的剪藏文件
```

**第 2 步：查看 Frontmatter（3 秒）**

```
动作：展开 frontmatter
重点：指向 fragment_url 字段
```

**第 3 步：点击链接（2 秒）**

```
动作：Cmd/Ctrl + 点击 fragment_url 的值
```

**第 4 步：浏览器打开（5 秒）**

```
效果：
  1. 浏览器自动打开原网页
  2. 页面滚动到剪藏的位置
  3. 选中的内容被高亮显示（黄色背景）
```

**第 5 步：对比展示（7 秒）**

```
动作：在浏览器和 Obsidian 之间切换
效果：展示内容的一致性
```

## 🎬 场景 3：多次剪藏演示（40 秒）

### 目的

展示同一页面多次剪藏不会覆盖

### 录制步骤

**第 1 步：第一次剪藏（12 秒）**

```
1. 选中第一段文本
2. 右键剪藏
3. 添加评论："第一次剪藏"
4. 确认
5. 记录文件路径
```

**第 2 步：等待（2 秒）**

```
动作：等待 2-3 秒（确保时间戳不同）
```

**第 3 步：第二次剪藏（12 秒）**

```
1. 在同一页面选中另一段文本
2. 右键剪藏
3. 添加评论："第二次剪藏"
4. 确认
5. 记录文件路径
```

**第 4 步：在 Obsidian 中验证（14 秒）**

```
动作：切换到 Obsidian
效果：
  1. 显示两个不同的文件
  2. 文件名包含不同的时间戳
  3. 打开两个文件，展示不同的内容和评论
```
