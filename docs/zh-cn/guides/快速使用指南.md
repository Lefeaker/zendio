1. **解压文件**

   - 打开 Chrome 浏览器
   - 地址栏输入：`chrome://extensions/`

### 如何更换 Logo

如果将来需要更换 logo：

```bash
# 1. 准备新的 logo 文件（建议 256x256 或更大）
# 2. 生成不同尺寸
sips -z 16 16 新logo.png --out assets/icons/icon16.png
sips -z 48 48 新logo.png --out assets/icons/icon48.png
sips -z 128 128 新logo.png --out assets/icons/icon128.png
cp 新logo.png assets/logo/options-logo.png

# 3. 重新构建
npm run build
```

## 🔄 版本更新流程

### 1. 修改版本号

编辑 `src/manifest.json`：
```json
{
  "version": "0.2.0"
}
```

### 2. 生成新版本

```bash
npm run release
```

### 3. 分发新版本

将新生成的 zip 文件发送给用户。

### 4. 用户更新方法

**方法一：覆盖更新**
- 用新版本文件覆盖旧的 extension 文件夹
- 在 `chrome://extensions/` 点击"重新加载"

**方法二：重新安装**
- 移除旧版本
- 加载新版本

## ⚠️ 注意事项

### 开发者注意

## 🆘 常见问题

### Q: 扩展图标不显示？
A: 检查 `dist/assets/` 目录下是否有图标文件，运行 `npm run build` 重新构建。

### Q: 朋友说无法安装？
A: 确认他们：
1. 开启了开发者模式
2. 选择的是 `extension` 文件夹（不是外层文件夹）
3. Chrome 版本较新

### Q: 如何测试扩展？
A: 
1. 运行 `npm run build`
2. 在 Chrome 中加载 `dist` 文件夹
3. 测试功能
4. 查看控制台错误（F12）

### Q: 扩展突然不工作了？
A: 
1. 在 `chrome://extensions/` 点击"重新加载"
2. 检查是否有错误信息
3. 重新运行 `npm run build`
