## 📌 5分钟快速上手

### 第1步：重新加载扩展（必须！）

由于代码已更新，你需要重新加载扩展：

1. 打开 Chrome 浏览器
2. 访问 `chrome://extensions/`
3. 找到 "All in Ob" 扩展
4. 点击 🔄 **刷新** 按钮

> ⚠️ **重要**：如果不重新加载，新功能将不会生效！

### 第2步：测试通义千问

1. **访问通义千问**
   ```
   https://tongyi.aliyun.com
   ```

2. **登录并创建对话**
   - 输入任意问题，例如："你好，请介绍一下自己"

3. **剪藏对话**
   - 点击浏览器工具栏中的扩展图标
   - 等待几秒钟

4. **查看结果**
   - 打开 Obsidian
   - 查看 `AI/tongyi/2025/` 目录
   - 找到新创建的 Markdown 文件

### 第3步：验证功能

打开导出的文件，检查：
- ✅ 文件包含完整的对话内容
- ✅ 用户和助手的消息都正确标记

## ❓ 常见问题

### Q1: 页面被识别为文章而非AI聊天？

**A**: 确保你已经重新加载了扩展！

```bash
# 如果还是不行，重新编译
cd AiiinOB/your-extension
npm run build
```

然后在 Chrome 中重新加载扩展。

### Q2: 没有提取到对话内容？

**A**: 可能的原因：
1. 页面还在加载中 - 等待几秒后再试
2. 页面结构已更新 - 查看控制台错误信息
3. 需要滚动加载 - 滚动到对话底部后再剪藏

### Q3: 文件保存到哪里了？

**A**: 默认路径：
- 通义千问：`AI/tongyi/2025/2025-10-02_对话标题.md`
- DeepSeek：`AI/deepseek/2025/2025-10-02_对话标题.md`
- Kimi：`AI/kimi/2025/2025-10-02_对话标题.md`

可以在扩展选项中自定义路径模板。

### Q4: 如何自定义保存路径？

**A**: 
1. 右键扩展图标 → 选项
2. 找到"路径模板"设置
3. 修改 AI 对话模板，例如：
   ```
   AI/{platform}/{yyyy}/{mm}/{title}.md
   ```

### Q5: 支持哪些内容格式？

**A**: 支持以下格式：
- ✅ 代码块（带语法高亮）
- ✅ 列表（有序/无序/嵌套）
- ✅ 表格
- ✅ 链接和图片
- ✅ 引用块
- ✅ 粗体和斜体
