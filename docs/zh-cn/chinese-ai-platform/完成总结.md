## 🐛 重要修复

### AI聊天页面识别

**问题**：中国AI平台的页面被识别为文章而非AI聊天

**原因**：`src/content/detect.ts` 中的 `isAIChat` 函数没有包含中国平台的URL

**解决**：

```typescript
// 添加中国AI平台的URL检测
if (/(tongyi\.aliyun\.com|chat\.deepseek\.com|kimi\.moonshot\.cn)/i.test(url)) return true;
```

**影响**：这是一个关键修复，确保了中国平台能被正确识别和处理
