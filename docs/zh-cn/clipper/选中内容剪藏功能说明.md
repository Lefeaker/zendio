# 📎 选中内容剪藏功能说明

## 功能概述

这是一个全新的**碎片化剪藏**功能，允许你快速收集网页中的特定内容片段，而不是整个页面。这个功能特别适合：

- 📝 收集有价值的段落或引用
- 💡 保存灵感片段
- 🔖 标记重要信息
- 💭 添加个人评论和思考

## 核心特性

### 1. 精确定位链接 🎯

使用 **Text Fragment** 技术，生成的链接可以直接定位到网页中的特定内容位置，类似于 OpenAI Deep Research 的效果。

示例链接格式：
```
https://example.com/article#:~:text=这是选中的文本内容
```

点击这个链接会：
- 打开原网页
- 自动滚动到选中的内容
- 高亮显示该内容（浏览器支持的情况下）

### 2. 添加个人评论 💭

在剪藏前，会弹出一个优雅的对话框，让你可以：
- 预览选中的内容
- 添加你的想法、笔记或评论
- 选择是否继续剪藏

### 3. 简洁的 Markdown 格式 📄

生成的 Markdown 文件包含：

```markdown
---
type: clipper
title: "页面标题 - 内容预览... @142345"
page_title: "页面标题"
url: "原始URL"
fragment_url: "精确定位URL"
clipped_at: "2025-10-02T14:23:45"
tags: [clipping, fragment]
---

这里是选中的内容转换成的 Markdown...

---

## 💭 我的评论

这里是你添加的评论内容...
```

**设计理念**：
- ✅ **简洁优先**：去除冗余信息，所有元数据都在 YAML frontmatter 中
- ✅ **内容为主**：剪藏的内容直接展示，无需额外标题
- ✅ **评论在下**：个人思考放在内容下方，符合阅读习惯
- ✅ **易于编辑**：简洁的格式便于后续编辑和整理

## 使用方法

### 方式一：右键菜单（推荐）

1. 在网页上选中你想要剪藏的文本
2. 右键点击选中的文本
3. 选择 **"✂️ 剪藏选中内容到 Obsidian"**
4. 在弹出的对话框中：
   - 查看选中内容的预览
   - （可选）添加你的评论
   - 点击 **"✂️ 剪藏"** 按钮
5. 等待通知确认保存成功

### 方式二：完整页面剪藏

如果你想剪藏整个页面（原有功能），可以：
- 右键点击页面空白处
- 选择 **"剪藏整个页面到 Obsidian"**

## 与其他功能的区别

| 功能 | 适用场景 | 内容类型 | 特点 |
|------|---------|---------|------|
| **选中内容剪藏** | 碎片化收集 | 部分内容 | 精确定位、可添加评论、快速 |
| **文章剪藏** | 完整阅读 | 整篇文章 | 完整内容、结构化 |
| **AI 对话剪藏** | 对话记录 | AI 聊天 | 对话格式、多轮交互 |

## 技术细节

### Text Fragment URL

这个功能使用了 Web 标准的 [Text Fragments](https://web.dev/text-fragments/) 技术：

- ✅ Chrome/Edge 80+ 原生支持
- ✅ Safari 16.1+ 支持
- ✅ Firefox 需要扩展支持

### 数据流程

```
选中文本 → 右键菜单 → 显示对话框 → 添加评论 → 
提取 HTML → 转换 Markdown → 生成 Fragment URL → 
保存到 Obsidian → 显示通知
```

## 最佳实践

### 1. 选择合适的内容长度

- ✅ **推荐**：1-3 个段落
- ⚠️ **可以**：单个句子或较长段落
- ❌ **不推荐**：整个页面（请使用完整页面剪藏）

### 2. 添加有价值的评论

好的评论示例：
```
💡 这个观点很有启发性，可以应用到我的项目中
🤔 需要进一步验证这个数据的来源
📌 与《XXX》一书中的观点相呼应
```

### 3. 使用标签组织

在 Obsidian 中，你可以：
- 使用 `#clipping` 标签查找所有剪藏
- 使用 `#fragment` 标签查找选中内容剪藏
- 添加自定义标签进行分类

### 4. 利用精确定位链接

在 Obsidian 中点击 "精确定位" 链接：
- 会在浏览器中打开原网页
- 自动滚动到你剪藏的内容
- 方便回顾上下文

## 故障排除

### 问题：对话框没有显示

**解决方案**：
- 确保你选中了文本
- 刷新页面后重试
- 检查浏览器控制台是否有错误

### 问题：精确定位链接无法工作

**原因**：
- 网页内容可能已更新
- 浏览器不支持 Text Fragments
- 网站使用了动态加载内容

**解决方案**：
- 使用 "原文链接" 访问页面
- 手动搜索内容

### 问题：保存失败

**检查**：
- Obsidian Local REST API 是否运行
- 网络连接是否正常
- 查看浏览器通知中的错误信息
