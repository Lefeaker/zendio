# 更新总结 - 片段剪藏增强功能

## 📅 更新日期
2025-10-03

## 🎯 更新目标

为片段剪藏功能添加两个重要特性：
1. **脚注格式支持** - 兼容 Obsidian Sidebar Highlights 插件
2. **上下文捕捉** - 可选择捕捉选中文字周围的上下文并高亮标记

## ✅ 完成的功能

> 最新说明：选项页已取消“上下文长度/模式”输入框，开启“捕捉上下文”后系统会自动获取约 200 字的前后段落，无需额外设置。

### 1. 脚注格式支持

**功能描述：**
- 评论以 Obsidian 脚注格式保存
- 完全兼容 Sidebar Highlights 插件
- 支持多行评论自动缩进

**格式示例：**
```markdown
这是选中的文本内容[^1]

[^1]: 这是我的评论
```

**优势：**
- ✅ 更简洁，无分隔线和标题
- ✅ 兼容 Sidebar Highlights 插件
- ✅ 评论紧跟内容，阅读体验更好
- ✅ 支持多行评论（自动缩进）

### 2. 上下文捕捉

**功能描述：**
- 可选择是否捕捉选中文字周围的上下文
- 用 `==高亮==` 标记实际选中的部分
- 在 Obsidian 中显示为黄色高亮

**格式示例：**
```markdown
前面的上下文内容 ==这是选中的文本== 后面的上下文内容
```

**配置选项：**
- **上下文长度**：50-1000 字符（默认 200）
- **上下文模式**：按字符数 / 按句子数

### 3. 设置界面

**新增配置区块：**
- ✅ 使用脚注格式（复选框）
- ✅ 捕捉上下文（复选框）
- ✅ 上下文长度（数字输入）
- ✅ 上下文模式（下拉选择）
- ✅ 示例展示
- ✅ 动态显示/隐藏上下文选项

## 🔧 技术实现细节

### 上下文提取算法

**按字符数模式：**
```typescript
const contextLen = config.contextLength;
beforeText = bodyText.substring(
  Math.max(0, selectedStart - contextLen),
  selectedStart
);
afterText = bodyText.substring(
  selectedEnd,
  Math.min(bodyText.length, selectedEnd + contextLen)
);
```

**按句子数模式：**
```typescript
// 1. 提取周围文本（前后各 1000 字符）
const surroundingText = bodyText.substring(
  Math.max(0, selectedStart - 1000),
  Math.min(bodyText.length, selectedEnd + 1000)
);

// 2. 分割句子
const sentences = surroundingText.split(/([.!?。！？]+\s*)/);

// 3. 找到包含选中文本的句子
// 4. 提取前后 N 个句子
```

### 脚注格式生成

**单行评论：**
```markdown
文本内容[^1]

[^1]: 评论内容
```

**多行评论：**
```markdown
文本内容[^1]

[^1]: 第一行
    第二行（自动缩进 4 空格）
    第三行（自动缩进 4 空格）
```

**实现代码：**
```typescript
const commentLines = userComment.trim().split('\n');
const formattedComment = commentLines
  .map((line, index) => index === 0 ? line : `    ${line}`)
  .join('\n');

markdown += `[^1]\n\n[^1]: ${formattedComment}`;
```

### 高亮标记生成

```typescript
if (context && clipperConfig.captureContext) {
  const beforeText = context.before.replace(/\s+/g, ' ').trim();
  const afterText = context.after.replace(/\s+/g, ' ').trim();
  
  const parts: string[] = [];
  if (beforeText) parts.push(beforeText);
  parts.push(`==${context.selected}==`);
  if (afterText) parts.push(afterText);
  
  contentMd = parts.join(' ');
}
```
