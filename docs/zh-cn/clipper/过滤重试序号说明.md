# 过滤 ChatGPT 重试序号说明

## 🎯 问题描述

ChatGPT 在重新生成回复时会显示重试序号，例如：

```html
<div class="px-0.5 text-sm font-semibold tabular-nums">2/2</div>
```

这些序号会出现在剪藏的内容中，造成干扰：

```markdown
# 2 GPT-4

2/2

这是 AI 的回复内容...
```

---

## ✅ 解决方案

### 方法 1：通过 CSS 类过滤

删除所有包含 `tabular-nums` 类的元素：

```typescript
const selectorsToRemove = [
  // ...
  '.tabular-nums',
  '[class*="tabular-nums"]'
];
```

**优点**：

- 精确定位
- 不会误删其他内容

**覆盖范围**：

- `class="tabular-nums"`
- `class="text-sm font-semibold tabular-nums"`
- 任何包含 `tabular-nums` 的 class

### 方法 2：通过文本模式过滤

删除匹配 `数字/数字` 模式的文本：

```typescript
const textPatternsToRemove = [
  // ...
  /^\d+\/\d+$/ // 匹配 "1/2", "2/2", "3/5" 等
];
```

**优点**：

- 通用性强
- 不依赖 CSS 类

**匹配示例**：

- ✅ `1/2`
- ✅ `2/2`
- ✅ `10/15`
- ❌ `1/2 完成`（不匹配，因为有额外文本）
- ❌ `版本 1/2`（不匹配，因为有前缀）

## 🔍 调试方法

### 检查是否有重试序号

在 ChatGPT 页面的 Console 中运行：

```javascript
// 查找所有 tabular-nums 元素
const retryIndicators = document.querySelectorAll('.tabular-nums, [class*="tabular-nums"]');
console.log('找到的重试序号元素:', retryIndicators.length);

retryIndicators.forEach((el, i) => {
  console.log(`元素 ${i + 1}:`, {
    text: el.textContent,
    class: el.className,
    html: el.outerHTML.substring(0, 200)
  });
});
```

### 检查是否被正确过滤

剪藏后，在生成的文件中搜索：

```
1. 搜索 "/2"
2. 搜索 "/3"
3. 搜索 "tabular"
```

如果找到这些内容，说明过滤失败。

## 🎨 其他可能的重试序号格式

ChatGPT 可能使用不同的格式显示重试序号：

### 格式 1：纯数字

```html
<div class="tabular-nums">2/2</div>
```

**过滤**：✅ 已支持

### 格式 2：带图标

```html
<div class="tabular-nums">
  <svg>...</svg>
  2/2
</div>
```

**过滤**：✅ 已支持（删除整个 div）

### 格式 3：带文字

```html
<div>重试 2/2</div>
```

**过滤**：⚠️ 部分支持（如果没有 `tabular-nums` 类，需要额外处理）

### 格式 4：其他位置

```html
<span class="retry-count">2/2</span>
```

**过滤**：⚠️ 需要添加 `.retry-count` 选择器

## 🔧 扩展过滤规则

如果发现新的重试序号格式，可以添加：

### 添加 CSS 选择器

在 `selectorsToRemove` 数组中添加：

```typescript
const selectorsToRemove = [
  // ...
  '.retry-count',
  '[class*="retry"]',
  '[data-testid="retry-indicator"]'
];
```

### 添加文本模式

在 `textPatternsToRemove` 数组中添加：

```typescript
const textPatternsToRemove = [
  // ...
  /^重试\s*\d+\/\d+$/, // "重试 2/2"
  /^Retry\s*\d+\/\d+$/i, // "Retry 2/2"
  /^\d+\s*of\s*\d+$/i // "2 of 2"
];
```

## 🐛 已知限制

### 限制 1：内容中的分数

如果 AI 回复中包含分数，可能会被误删：

**示例**：

```
这个方案的评分是 8/10
```

**解决方案**：

- 当前的正则 `/^\d+\/\d+$/` 只匹配**整行**是数字/数字的情况
- 所以 "这个方案的评分是 8/10" 不会被删除 ✅

### 限制 2：特殊格式的重试序号

如果 ChatGPT 使用新的格式，可能无法识别：

**示例**：

```html
<div class="new-retry-class">Response 2 of 2</div>
```

**解决方案**：

- 发现新格式后，添加对应的选择器或正则
